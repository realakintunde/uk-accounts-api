## How to Use Each Feature

### Getting Started with Teams

1. **Add Team Members:**
   - Go to Settings → Team Members
   - Click "Invite Member"
   - Enter their email address
   - Select their role (Admin, Accountant, Viewer, Approver)
   - They'll receive an invitation

2. **Manage Roles:**
   - Each role has different permissions
   - **Admin**: Full access to everything
   - **Accountant**: Create/edit financial data
   - **Viewer**: Read-only access to reports
   - **Approver**: Can approve document submissions

### Document Versioning

1. **Create a Version:**
   - Edit a statement or document
   - Click "Save as New Version"
   - Add a summary of changes (e.g., "Updated revenue figures")
   - System saves the version automatically

2. **View Version History:**
   - Open a document
   - Click "Version History"
   - See all previous versions
   - Click any version to view it

3. **Compare Versions:**
   - Select two versions to compare
   - See exactly what changed
   - View changes highlighted

4. **Revert to Previous Version:**
   - Open version history
   - Click "Restore" on the version you want
   - Creates a new version with that content

### Approval Workflows

1. **Request Approval:**
   - Edit your financial statement
   - Click "Request Approval"
   - Select who needs to approve (can select multiple people)
   - Add optional instructions
   - Submit for approval

2. **Track Approvals:**
   - See each step of the approval process
   - View status: Pending, Approved, or Rejected
   - See who needs to approve next
   - Check approver comments

3. **Approve as Reviewer:**
   - Check your dashboard
   - You'll see "Pending Approvals" section
   - Click to review the document
   - Click "Approve" with optional comment
   - Or click "Request Changes" with feedback

### Managing Budgets

1. **Create Annual Budget:**
   - Go to Financial → Budgets
   - Click "New Budget"
   - Enter fiscal year (e.g., 2026)
   - Add budget items:
     - Salaries: $200,000
     - Operations: $50,000
     - Marketing: $30,000
   - Total: $280,000

2. **Track Spending:**
   - System automatically tracks actual spending
   - See Variance % (negative = under budget)
   - Identify overspending early
   - Adjust spending targets

### Bank Reconciliation

1. **Connect Bank Account:**
   - Go to Financial → Bank Accounts
   - Click "Add Bank Account"
   - Enter account details:
     - Account name (e.g., "Main Checking")
     - Account number
     - Sort code
     - Bank name

2. **Import Transactions:**
   - Download transactions from your bank (CSV or OFX)
   - Upload to the system
   - System imports automatically

3. **Reconcile:**
   - Compare bank statement with imported transactions
   - Check off each transaction as it matches
   - System calculates difference
   - When difference = 0, you're reconciled!

### Setting Up Recurring Entries

1. **Add Recurring Entry:**
   - Go to Financial → Recurring Entries
   - Click "New Entry"
   - Enter details:
     - Description (e.g., "Monthly rent")
     - Amount ($5,000)
     - Category (Rent)
     - Frequency (Monthly/Quarterly/Annually)
     - Start date and optional end date

2. **System Automatically:**
   - Creates entries on schedule
   - Reduces manual work
   - Ensures consistency
   - Great for fixed expenses

### Configuring Tax Settings

1. **Set Tax Parameters:**
   - Go to Settings → Tax Settings
   - Enter tax year end date (e.g., March 31)
   - Set VAT rate (default: 20%)
   - Set corporation tax rate (default: 19%)
   - Set payroll tax rate (default: 8%)
   - Enter next tax return deadline

2. **Benefits:**
   - System applies rates automatically
   - Calculates tax liability correctly
   - Tracks tax deadlines
   - Prepares tax calculations

### Viewing Notifications

1. **Check Notifications:**
   - Click bell icon (top right)
   - See list of recent notifications
   - Unread count shows important items

2. **Notification Types:**
   - Document approval requested → URGENT
   - Team member added you → INFO
   - Comment added to document → ACTION
   - Submission approved → SUCCESS

3. **Manage Notifications:**
   - Click to mark as read
   - Delete old notifications
   - Set email preferences
   - Choose what alerts you get

### Viewing Audit Logs

1. **Access Audit Trail:**
   - Go to Settings → Audit Logs
   - See complete history of changes
   - Filter by action type, user, date
   - Export for compliance

2. **Audit Log Shows:**
   - WHO made the change (user email)
   - WHAT changed (field name)
   - WHEN it happened (timestamp)
   - OLD values and NEW values
   - WHY (change summary if provided)

## Best Practices

### Security
- Use strong passwords for all team members
- Review audit logs regularly
- Archive old documents
- Remove team members who leave

### Collaboration
- Add clear change summaries when versioning
- Use approval workflows for important documents
- Leave helpful comments for reviewers
- Maintain clear approval chains

### Financial Management
- Reconcile bank accounts monthly
- Review budgets quarterly
- Set up recurring entries for fixed costs
- Track tax deadlines

### Compliance
- Keep audit logs for 7+ years
- Document all changes
- Maintain approval trails
- Regular backups of all data

## Troubleshooting

**Q: Team member can't see company data?**
A: Check their role - Viewers have read-only access

**Q: How do I find old document versions?**
A: Open the document, click Version History

**Q: Approval workflow stuck?**
A: Check who's assigned - may need follow-up

**Q: Bank reconciliation not matching?**
A: Look for duplicates or missing transactions

**Q: Need to undo a change?**
A: Restore a previous document version
